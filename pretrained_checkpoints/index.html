Pretrained checkpoints

A starting point for your own training.

These checkpoints were trained on the small example datasets below. They're meant as a starting point to fine-tune on your own videos, not finished off-the-shelf classifiers. Each file embeds its training config, class names, and multi-label flag, so you can pass one to feral train and continue from there.

Run inference

feral infer ./ant_example.pt ./your_videos

Fine-tune from it

feral train ./your_videos ./your_labels.json -c ./ant_example.pt
